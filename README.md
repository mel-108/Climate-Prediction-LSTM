# 🌍 Climate Change Prediction using LSTM

## Project Description

An LSTM-based climate change prediction system that forecasts key environmental variables such as Carbon Dioxide (CO₂), Methane (CH₄), Solar Radiation, Sea Level, Precipitation, and Temperature Anomaly. The project includes an interactive user interface for analyzing historical data and generating future projections with scenario-based insights.

## Features

*  Predicts multiple climate variables
*  Uses LSTM (Deep Learning) for time-series forecasting
*  Displays actual vs predicted graphs
*  Interactive UI using Gradio
*  Scenario analysis (baseline, high, low)

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy, Pandas
* Matplotlib
* Gradio
